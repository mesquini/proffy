## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/mesquini/proffy

> cd proffy

> cd server
> yarn install
> yarn dev

> cd web
> yarn install
> yarn start

> cd mobile
> yarn install
> yarn start
```
